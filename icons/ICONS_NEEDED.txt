IMPORTANT: You need to create actual PNG icons before publishing this extension.

Required icons:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

Design suggestions:
- Use the extension's purple/blue gradient theme (#667eea to #764ba2)
- Create a simple, recognizable logo (e.g., stylized "G" or stacked layers icon)
- Ensure the icon is visible at small sizes
- Use transparent background

Tools to create icons:
1. Figma (free online design tool)
2. Canva (free with templates)
3. GIMP (free desktop app)
4. Photoshop (paid)
5. Online icon generators

Quick option:
You can use a favicon generator like:
- https://favicon.io/
- https://www.favicon-generator.org/

Just create a 128x128 PNG and let the tool generate smaller sizes.
